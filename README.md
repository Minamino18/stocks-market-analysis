## Overview

This project is an end-to-end stock market analysis application built using Python. It demonstrates real-world data analytics skills, including data sourcing, cleaning, statistical analysis, machine learning, and interactive visualization using near real-time financial data.

## Features

- Near real-time stock price updates
- Moving average crossover analysis (50-day & 200-day)
- Risk vs return analysis
- Sharpe Ratio and Beta calculation
- Correlation heatmap for diversification insights
- Machine learning price trend estimation
- Interactive Streamlit dashboard
- Downloadable CSV financial reports

## Technology Stack

- Python
- Pandas & NumPy
- yfinance (Yahoo Finance API)
- Plotly
- Streamlit
- Scikit-learn
